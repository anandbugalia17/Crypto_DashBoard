# CryptoDash - Cryptocurrency Dashboard

A responsive cryptocurrency dashboard built with React and Tailwind CSS.

## Features

- Header with branding
- Search bar (UI only)
- 4 statistic cards: Bitcoin Price, Market Cap, 24h Change, Trading Volume
- Table displaying 5 sample cryptocurrencies with hardcoded data
- Fully responsive layout (mobile, tablet, desktop)

## Tech Stack

- React (Vite)
- Tailwind CSS v4
